Vigencia de los datos

Pathwise utiliza un archivo de base de datos de tipo SQLite v3.x, que es generado por el servidor de datos (expofinder.uma.es). Hay dos formas de trabajar con dicho archivo.

  1. Generando uno "al vuelo", con la información que está conservada en el servidor en el momento de realizar la solicitud. Dicho archivo es descargado directamente en el ordenador cliente y abierto desde allí para poder trabajar con él.
  2. Utilizando una "copia estática", que el servidor realiza periódicamente. En este caso, el archivo no se descarga, sino que se abre de forma remota.

El método A permite conservar el archivo a modo de "foto fija" del estado de ExpoFinder en un momento dado, y facilita la realización de comparaciones.
