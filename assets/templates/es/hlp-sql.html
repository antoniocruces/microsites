SQL

Microsites (MS) usa la biblioteca de código SQL = Structured Query Language .js, escrita por kripken alias de Alon Zakai, ingeniero de software que trabajó en Google y Mozilla con tecnología WASM = Web Assembly , que utiliza subconjunto de reglas ANSI/SQL. Con ellas puede construir cuantas consultas desee. Los resultados son exportables.

SQL.js Alon Zakai WASM ANSI/SQL SQL en SQLite

Parte de la información es una reelaboración de la ofrecida en GeoTalleres.

IMPORTANTE: Todas las operaciones de MS se realizan en la memoria volátil del ordenador cliente. En ningún momento la base de datos cargada se verá afectada, pero cualquier alteración que realice en las tablas del sistema puede afectar a los resultados obtenidos durante la sesión de trabajo actual.

Tablas y campos

Leyenda: INTEGER = número entero; VARCHAR = texto de longitud variable; DATE = fecha; NOT NULL = no acepta valores nulos.
Prefijos: start = inicial; end = final; age = edad.
Sufijos: y = año; m = mes; d = día; w = semana. Campos utilitarios: ID = número ID; RID = número ID relacionado; rtype/idtype = tipo de registro; ridtype = tipo de registro relacionado; rkey = clave de registro; value = valor de registro.
