Carga

Microsites utiliza un archivo de base de datos en formato SQLite v3, con una estructura específica, generado por ExpoFinder. Se carga en memoria solamente una vez durante cada sesión de trabajo. Si se selecciona una colección distinta, Microsites construirá un nuevo subconjunto de registros. No se volverá a cargar la base de datos, pero se realizará un nuevo recuento. Si desea ayuda sobre el proceso de carga y recuento, pulse aquí.
